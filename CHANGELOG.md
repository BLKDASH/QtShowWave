# SSW Serial Helper 版本记录

## 当前版本：v2.0.1

---

## 已有功能

### 串口通信
- 串口自动检测与列表刷新
- 支持常用波特率配置（115200 等）
- 支持数据位、停止位、校验位配置
- 独立工作线程处理串口收发，不阻塞 UI
- 自动重连上次使用的串口

### 数据接收与显示
- ASCII / 十六进制双模式显示
- 实时速度监控（bytes/s）与累计字节统计
- 时间戳显示（HH:mm:ss.zzz 格式）
- 自动滚动与手动暂停滚动
- 关键词高亮（日志级别：DEBUG/INFO/WARN/ERROR/FATAL 等）
- 十六进制模式下 0A/0D 换行支持

### 数据发送
- 普通文本发送
- 十六进制发送
- 自动追加换行符选项
- 发送后自动清空选项

### 编码支持
- ANSI
- UTF-8
- GBK

### 界面与设置
- 深色模式
- 字体大小调节（6-24pt）
- 多字体选择（系统默认 / HarmonyOS Sans SC / Alibaba PuHuiTi / MiSans）
- 窗口大小与分割条位置记忆
- 所有设置自动持久化保存
